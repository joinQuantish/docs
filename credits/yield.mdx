---
title: "Yield"
description: "Interest paid on Polymarket balances — daily on-chain distributions since July 2025."
---

Polymarket pays yield on eligible USD balances. It lands as one daily on-chain batch (around 00:10 UTC) to every enrolled wallet. PolyNode indexes the full history: **$2.74M to 321,307 wallets since 2025-07-17** — the largest credit type by recipient count.

All credits endpoints take this type as `yield`.

## Lifetime total for a wallet

```bash
curl "https://api.polynode.dev/v3/wallets/0x204f72f35326db932158cba6adff0b9a1da95e14/credits/totals?types=yield" \
  -H "x-api-key: $POLYNODE_API_KEY"
```

## This month's earnings for a wallet

Add unix-second bounds for windowed totals:

```bash
curl "https://api.polynode.dev/v3/wallets/0x204f72f35326db932158cba6adff0b9a1da95e14/credits/totals?types=yield&start_ts=1783036800&end_ts=1785715200" \
  -H "x-api-key: $POLYNODE_API_KEY"
```

## Yield leaderboard

```bash
curl "https://api.polynode.dev/v3/credits/leaderboard?type=yield&limit=25" \
  -H "x-api-key: $POLYNODE_API_KEY"
```

Windows: `window=24h|7d|30d|all`, or explicit `start_ts`/`end_ts`.

## Payout history for a wallet

Filterable by time and amount, sortable by `time` or `amount`:

```bash
curl "https://api.polynode.dev/v3/wallets/0x204f72f35326db932158cba6adff0b9a1da95e14/credits?types=yield&min_amount=100&sort=amount&order=desc&limit=50" \
  -H "x-api-key: $POLYNODE_API_KEY"
```

## Earnings chart data

```bash
curl "https://api.polynode.dev/v3/wallets/0x204f72f35326db932158cba6adff0b9a1da95e14/credits/series?types=yield&interval=week&fill=zero" \
  -H "x-api-key: $POLYNODE_API_KEY"
```

## Watch payouts land globally

```bash
curl "https://api.polynode.dev/v3/credits/events?types=yield&min_amount=1000&limit=50" \
  -H "x-api-key: $POLYNODE_API_KEY"
```

```python
import requests

BASE = "https://api.polynode.dev"
HEADERS = {"x-api-key": "YOUR_KEY"}

wallet = "0x204f72f35326db932158cba6adff0b9a1da95e14"
totals = requests.get(
    f"{BASE}/v3/wallets/{wallet}/credits/totals",
    params={"types": "yield"}, headers=HEADERS,
).json()
print(totals["totals"]["yield"]["amount"])
```
