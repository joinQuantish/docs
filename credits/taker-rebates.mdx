---
title: "Taker Rebates"
description: "Taker fee rebates — volume-based rebates to active takers, launched June 2026."
---

Polymarket's newest rebate program returns part of taker fees to active traders. Payouts run in on-chain batches and include large one-off settlements (single credits above $1M exist). PolyNode indexes all of it: **$13.7M to 36,469 wallets since 2026-06-20**.

All credits endpoints take this type as `taker_rebate`.

## Lifetime total for a wallet

```bash
curl "https://api.polynode.dev/v3/wallets/0x204f72f35326db932158cba6adff0b9a1da95e14/credits/totals?types=taker_rebate" \
  -H "x-api-key: $POLYNODE_API_KEY"
```

## This month's earnings for a wallet

Add unix-second bounds for windowed totals:

```bash
curl "https://api.polynode.dev/v3/wallets/0x204f72f35326db932158cba6adff0b9a1da95e14/credits/totals?types=taker_rebate&start_ts=1783036800&end_ts=1785715200" \
  -H "x-api-key: $POLYNODE_API_KEY"
```

## Taker Rebates leaderboard

```bash
curl "https://api.polynode.dev/v3/credits/leaderboard?type=taker_rebate&limit=25" \
  -H "x-api-key: $POLYNODE_API_KEY"
```

Windows: `window=24h|7d|30d|all`, or explicit `start_ts`/`end_ts`.

## Payout history for a wallet

Filterable by time and amount, sortable by `time` or `amount`:

```bash
curl "https://api.polynode.dev/v3/wallets/0x204f72f35326db932158cba6adff0b9a1da95e14/credits?types=taker_rebate&min_amount=100&sort=amount&order=desc&limit=50" \
  -H "x-api-key: $POLYNODE_API_KEY"
```

## Earnings chart data

```bash
curl "https://api.polynode.dev/v3/wallets/0x204f72f35326db932158cba6adff0b9a1da95e14/credits/series?types=taker_rebate&interval=week&fill=zero" \
```

## Watch payouts land globally

```bash
curl "https://api.polynode.dev/v3/credits/events?types=taker_rebate&min_amount=1000&limit=50" \
  -H "x-api-key: $POLYNODE_API_KEY"
```

```python
import requests

BASE = "https://api.polynode.dev"
HEADERS = {"x-api-key": "YOUR_KEY"}

wallet = "0x204f72f35326db932158cba6adff0b9a1da95e14"
totals = requests.get(
    f"{BASE}/v3/wallets/{wallet}/credits/totals",
    params={"types": "taker_rebate"}, headers=HEADERS,
).json()
print(totals["totals"]["taker_rebate"]["amount"])
```
