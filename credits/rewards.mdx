---
title: "Liquidity Rewards"
description: "Polymarket's liquidity rewards program — daily payouts to accounts that quote rewarded markets inside the spread."
---

Polymarket pays daily liquidity rewards to makers who keep orders inside the max spread on rewarded markets (see Polymarket's Liquidity Rewards program docs). Each day's earnings arrive as an on-chain batch transfer. PolyNode indexes every payout ever made: **$28.1M to 174,850 wallets since 2023-11-25**.

<Note>
Reward market configurations (which markets pay rewards, rates and spreads) are served by the [Rewards](/data/rewards/markets) endpoints; this section covers the payouts themselves.
</Note>

All credits endpoints take this type as `reward`.

## Lifetime total for a wallet

```bash
curl "https://api.polynode.dev/v3/wallets/0x204f72f35326db932158cba6adff0b9a1da95e14/credits/totals?types=reward" \
  -H "x-api-key: $POLYNODE_API_KEY"
```

## This month's earnings for a wallet

Add unix-second bounds for windowed totals:

```bash
curl "https://api.polynode.dev/v3/wallets/0x204f72f35326db932158cba6adff0b9a1da95e14/credits/totals?types=reward&start_ts=1783036800&end_ts=1785715200" \
  -H "x-api-key: $POLYNODE_API_KEY"
```

## Liquidity Rewards leaderboard

```bash
curl "https://api.polynode.dev/v3/credits/leaderboard?type=reward&limit=25" \
  -H "x-api-key: $POLYNODE_API_KEY"
```

Windows: `window=24h|7d|30d|all`, or explicit `start_ts`/`end_ts`.

## Payout history for a wallet

Filterable by time and amount, sortable by `time` or `amount`:

```bash
curl "https://api.polynode.dev/v3/wallets/0x204f72f35326db932158cba6adff0b9a1da95e14/credits?types=reward&min_amount=100&sort=amount&order=desc&limit=50" \
  -H "x-api-key: $POLYNODE_API_KEY"
```

## Earnings chart data

```bash
curl "https://api.polynode.dev/v3/wallets/0x204f72f35326db932158cba6adff0b9a1da95e14/credits/series?types=reward&interval=week&fill=zero" \
  -H "x-api-key: $POLYNODE_API_KEY"
```

## Watch payouts land globally

```bash
curl "https://api.polynode.dev/v3/credits/events?types=reward&min_amount=1000&limit=50" \
  -H "x-api-key: $POLYNODE_API_KEY"
```

```python
import requests

BASE = "https://api.polynode.dev"
HEADERS = {"x-api-key": "YOUR_KEY"}

wallet = "0x204f72f35326db932158cba6adff0b9a1da95e14"
totals = requests.get(
    f"{BASE}/v3/wallets/{wallet}/credits/totals",
    params={"types": "reward"}, headers=HEADERS,
).json()
print(totals["totals"]["reward"]["amount"])
```

See the [Credits API Reference](/credits/wallet-totals) for every parameter.
