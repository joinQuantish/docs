# PolyNode API

PolyNode is a real-time data API for Polymarket. It detects settlement transactions **~2 seconds before on-chain confirmation** and serves them alongside OHLCV candles and enriched market metadata.

## What makes PolyNode different

<CardGroup cols={2}>

## How it works

PolyNode monitors the Polygon network for Polymarket activity and processes it in real time. Settlement transactions are detected before they confirm on-chain, enriched with market metadata, and delivered via REST API and WebSocket.

## Endpoints at a glance

| Endpoint | Description |
|----------|-------------|
| `GET /v1/markets` | Top markets by 24h volume |
| `GET /v1/markets/:token_id` | Market detail with metadata |
| `GET /v1/markets/slug/:slug` | Lookup by URL slug |
| `GET /v1/markets/condition/:id` | Lookup by condition ID |
| `GET /v1/markets/list` | Filtered + paginated listing |
| `GET /v1/search?q=` | Full-text market search |
| `GET /v1/stats/:token_id` | Market statistics + 24h OHLCV |
| `GET /v1/candles/:token_id` | OHLCV price candles |
| `GET /v1/status` | System status |
| `GET /v1/settlements/recent` | Recent settlements |
| `GET /v1/settlements/wallet/:addr` | Settlements by wallet |
| `GET /v1/settlements/token/:id` | Settlements by token |
| `GET /v1/wallets/:addr` | Wallet activity summary |
| `POST /v1/keys` | Generate API key |
| `GET /ws` | WebSocket streaming |
