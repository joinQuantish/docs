# PolyNode API

PolyNode is a real-time data API for Polymarket. It runs a full Polygon node, decodes settlement transactions from the **mempool** (before they confirm on-chain), and serves them alongside OHLCV candles and enriched market metadata.

## What makes PolyNode different

<CardGroup cols={2}>
  <Card title="Mempool data" icon="bolt">
    Decoded pending settlements ~2 seconds before on-chain confirmation. Nobody else offers this.
  </Card>
  <Card title="Combined data" icon="layer-group">
    Market metadata (question, slug, outcomes, images) enriched on every response. One call instead of separate APIs.
  </Card>
  <Card title="OHLCV candles" icon="chart-line">
    6 resolutions (1m, 5m, 15m, 1h, 4h, 1d) built from real settlement data.
  </Card>
  <Card title="Self-hostable" icon="server">
    Single Rust binary, under 2GB RAM. Run your own instance.
  </Card>
</CardGroup>

## Architecture

```
Polygon P2P Network
    │
    ▼
Bor Node (mempool monitoring)
    │
    ▼
Rust API Server ← Gamma API (metadata)
    │
    ├── Redis (settlements, indexes, pub/sub)
    ├── State Engine (in-memory markets, candles)
    └── Archive Writer (JSONL daily files)
    │
    ▼
REST API + WebSocket (your application)
```

## Endpoints at a glance

| Endpoint | Description |
|----------|-------------|
| `GET /v1/markets` | Top markets by 24h volume |
| `GET /v1/markets/:token_id` | Market detail with metadata |
| `GET /v1/markets/slug/:slug` | Lookup by URL slug |
| `GET /v1/markets/condition/:id` | Lookup by condition ID |
| `GET /v1/markets/list` | Filtered + paginated listing |
| `GET /v1/search?q=` | Full-text market search |
| `GET /v1/stats/:token_id` | Market statistics + 24h OHLCV |
| `GET /v1/candles/:token_id` | OHLCV price candles |
| `GET /v1/status` | System status |
| `GET /v1/settlements/recent` | Recent decoded settlements |
| `GET /v1/settlements/wallet/:addr` | Settlements by wallet |
| `GET /v1/settlements/token/:id` | Settlements by token |
| `POST /v1/keys` | Generate API key |
| `GET /ws` | WebSocket streaming |
