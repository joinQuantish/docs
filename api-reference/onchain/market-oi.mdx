---
title: "Market Open Interest"
description: "Onchain open interest for a specific market."
openapi: "GET /v2/onchain/markets/{condition_id}/oi"
---

Returns the current open interest for a specific market, sourced from onchain data.

## Request

```
GET /v2/onchain/markets/{condition_id}/oi
```

| Parameter | Type | Location | Description |
|-----------|------|----------|-------------|
| `condition_id` | string | path | Market condition ID (0x-prefixed, 64 hex chars) |

## Response

```json
{
  "condition_id": "0xa241c7f77a6e3918d37de906a8b89866dd27c46e6fa0d0de5624c5ee7fdc7ab6",
  "source": "onchain",
  "found": true,
  "open_interest_usdc": 803.52
}
```

| Field | Type | Description |
|-------|------|-------------|
| `found` | boolean | Whether the market was found onchain |
| `open_interest_usdc` | number | Current open interest in USDC |

## Example

```bash
curl "https://api.polynode.dev/v2/onchain/markets/0xa241c7f77a6e3918d37de906a8b89866dd27c46e6fa0d0de5624c5ee7fdc7ab6/oi" \
  -H "x-api-key: YOUR_KEY"
```
