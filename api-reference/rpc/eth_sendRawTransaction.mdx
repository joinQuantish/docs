## eth_sendRawTransaction

Submits a pre-signed transaction for broadcast to the Polygon network.

**Cache**: None (passthrough)

### Parameters

| Position | Type | Description |
|----------|------|-------------|
| 0 | `DATA` | Signed transaction data (hex-encoded) |

### Returns

`DATA` — 32-byte transaction hash, or error if the transaction was rejected.

### Example

```bash cURL
curl -s -X POST https://polynode.dev/rpc \
  -H "Content-Type: application/json" \
  -d '{
    "jsonrpc": "2.0",
    "method": "eth_sendRawTransaction",
    "params": ["0xf86c..."],
    "id": 1
  }'
```

### Response

```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": "0xe670ec64341771606e55d6b4ca35a1a6b75ee3d5145a99d05921026d1527331"
}
```
