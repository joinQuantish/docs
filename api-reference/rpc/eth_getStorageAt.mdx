---
title: "eth_getStorageAt"
description: "Returns the value of a storage slot at a given address."
---

<Warning>
**Coming Soon** — The JSON-RPC proxy is not yet live. PolyNode's core settlement streaming API is fully operational. RPC access will be enabled once our Polygon node sync completes.
</Warning>

## Parameters

| # | Type | Description |
|---|------|-------------|
| 1 | `string` | Contract address (20 bytes, hex-encoded) |
| 2 | `string` | Storage slot position (hex-encoded) |
| 3 | `string` | Block number (hex) or tag: `"latest"`, `"earliest"`, `"pending"` |

## Example

<CodeGroup>
```bash cURL
curl -X POST https://api.polynode.dev/rpc \
  -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","method":"eth_getStorageAt","params":["0x295a70b2de5e3953354a6a8344e616ed314d7251","0x0","latest"],"id":1}'
```
</CodeGroup>

## Response

```json
{
  "jsonrpc": "2.0",
  "result": "0x00000000000000000000000000000000000000000000000000000000000004d2",
  "id": 1
}
```

## Cache

**2 seconds** — storage can change every block.
