---
title: "eth_getCode"
description: "Returns the bytecode at a given contract address."
---

<Warning>
**Coming Soon** — The JSON-RPC proxy is not yet live. PolyNode's core settlement streaming API is fully operational. RPC access will be enabled once our Polygon node sync completes.
</Warning>

## Parameters

| # | Type | Description |
|---|------|-------------|
| 1 | `string` | Contract address (20 bytes, hex-encoded) |
| 2 | `string` | Block number (hex) or tag: `"latest"`, `"earliest"`, `"pending"` |

## Example

<CodeGroup>
```bash cURL
curl -X POST https://api.polynode.dev/rpc \
  -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","method":"eth_getCode","params":["0xa94f5374fce5edbc8e2a8697c15331677e6ebf0b","latest"],"id":1}'
```
</CodeGroup>

## Response

```json
{
  "jsonrpc": "2.0",
  "result": "0x600160008035811a818181146012578290509...",
  "id": 1
}
```

Returns `"0x"` for EOAs (non-contract addresses).

## Cache

**24 hours** — contract bytecode is immutable once deployed.
