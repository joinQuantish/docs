## pn_getWalletActivity

Returns a summary of recent activity for a wallet address, including trade count, volume, transfers, and deposits. Served from the in-memory state engine.

### Parameters

| Position | Type | Description |
|----------|------|-------------|
| 0 | `String` | Wallet address (0x...) |

### Returns

`Object` — wallet activity summary, or `null` if no activity found.

| Field | Type | Description |
|-------|------|-------------|
| `last_active` | number | Last activity timestamp (ms) |
| `trade_count` | number | Number of trades |
| `trade_volume_usd` | number | Total trade volume in USD |
| `transfer_count` | number | Number of token transfers |
| `deposit_count` | number | Number of deposits/withdrawals |
| `deposit_volume_usd` | number | Total deposit volume in USD |

### Example

```bash cURL
curl -s -X POST https://polynode.dev/rpc \
  -H "Content-Type: application/json" \
  -d '{
    "jsonrpc": "2.0",
    "method": "pn_getWalletActivity",
    "params": ["0x1234567890abcdef1234567890abcdef12345678"],
    "id": 1
  }'
```

### Response

```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": {
    "last_active": 1709136043000,
    "trade_count": 142,
    "trade_volume_usd": 58420.50,
    "transfer_count": 12,
    "deposit_count": 3,
    "deposit_volume_usd": 10000.00
  }
}
```
