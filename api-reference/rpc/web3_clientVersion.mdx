---
title: "web3_clientVersion"
description: "Returns the PolyNode client version string."
---

<Warning>
**Coming Soon** — The JSON-RPC proxy is not yet live. PolyNode's core settlement streaming API is fully operational. RPC access will be enabled once our Polygon node sync completes.
</Warning>

## Parameters

None.

## Example

<CodeGroup>
```bash cURL
curl -X POST https://api.polynode.dev/rpc \
  -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","method":"web3_clientVersion","params":[],"id":1}'
```
</CodeGroup>

## Response

```json
{
  "jsonrpc": "2.0",
  "result": "PolyNode/1.0.0/rust",
  "id": 1
}
```

Returns the PolyNode client identifier string.

## Cache

**Local** — no network call, instant response.
