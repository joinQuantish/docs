## eth_blockNumber

Returns the current block number on Polygon PoS.

**Cache**: 2 seconds

### Parameters

None.

### Returns

`QUANTITY` — hex-encoded current block number.

### Example

<CodeGroup>
```bash cURL
curl -s -X POST https://api.polynode.dev/rpc \
  -H "Content-Type: application/json" \
  -d '{
    "jsonrpc": "2.0",
    "method": "eth_blockNumber",
    "params": [],
    "id": 1
  }'
```

```bash wscat
wscat -c wss://api.polynode.dev/ws
> {"jsonrpc":"2.0","method":"eth_blockNumber","params":[],"id":1}
```
</CodeGroup>

### Response

```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": "0x5125D84"
}
```
