## eth_chainId

Returns the chain ID used for signing replay-protected transactions. Polygon PoS is chain ID `137` (`0x89`).

**Cache**: 24 hours (immutable)

### Parameters

None.

### Returns

`QUANTITY` — hex-encoded chain ID.

### Example

```bash cURL
curl -s -X POST https://api.polynode.dev/rpc \
  -H "Content-Type: application/json" \
  -d '{
    "jsonrpc": "2.0",
    "method": "eth_chainId",
    "params": [],
    "id": 1
  }'
```

### Response

```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": "0x89"
}
```
