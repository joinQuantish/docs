---
title: "eth_getBalance"
description: "Returns the balance of an account at a given block."
---

## Parameters

| # | Type | Description |
|---|------|-------------|
| 1 | `string` | Address to check (20 bytes, hex-encoded) |
| 2 | `string` | Block number (hex) or tag: `"latest"`, `"earliest"`, `"pending"` |

## Example

<CodeGroup>
```bash cURL
curl -X POST https://api.polynode.dev/rpc \
  -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","method":"eth_getBalance","params":["0x407d73d8a49eeb85d32cf465507dd71d507100c1","latest"],"id":1}'
```
</CodeGroup>

## Response

```json
{
  "jsonrpc": "2.0",
  "result": "0x0234c8a3397aab58",
  "id": 1
}
```

Result is the balance in wei, hex-encoded.

## Cache

**2 seconds** — balance can change every block (~2s on Polygon).
