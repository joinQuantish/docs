## pn_getMarketPrice

Returns the latest price, 24-hour volume, and trade count for a Polymarket token ID. Always live data with sub-millisecond response times.

### Parameters

| Position | Type | Description |
|----------|------|-------------|
| 0 | `String` | Polymarket token ID |

### Returns

`Object` — market price data, or `null` if token not found.

| Field | Type | Description |
|-------|------|-------------|
| `token_id` | string | Token ID |
| `last_price` | number | Last traded price (0-1 for binary markets) |
| `last_trade_at` | number | Timestamp of last trade (ms) |
| `volume_24h` | number | Rolling 24-hour volume in USD |
| `trade_count_24h` | number | Number of trades in last 24 hours |

### Example

```bash cURL
curl -s -X POST https://api.polynode.dev/rpc \
  -H "Content-Type: application/json" \
  -d '{
    "jsonrpc": "2.0",
    "method": "pn_getMarketPrice",
    "params": ["21742633143463906290569050155826241533067272736897614950488156847949938836455"],
    "id": 1
  }'
```

### Response

```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": {
    "token_id": "21742633143463906290569050155826241533067272736897614950488156847949938836455",
    "last_price": 0.72,
    "last_trade_at": 1772500000000,
    "volume_24h": 125340.50,
    "trade_count_24h": 847
  }
}
```
