## eth_getBlockByHash

Returns information about a block by its hash.

**Cache**: 30 seconds

### Parameters

| Position | Type | Description |
|----------|------|-------------|
| 0 | `DATA` | 32-byte block hash |
| 1 | `Boolean` | If `true`, returns full transaction objects; if `false`, only hashes |

### Returns

`Object` — block object, or `null` if not found.

### Example

```bash cURL
curl -s -X POST https://api.polynode.dev/rpc \
  -H "Content-Type: application/json" \
  -d '{
    "jsonrpc": "2.0",
    "method": "eth_getBlockByHash",
    "params": ["0xabc123...", false],
    "id": 1
  }'
```

### Response

```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": {
    "number": "0x5125d84",
    "hash": "0xabc123...",
    "parentHash": "0x...",
    "timestamp": "0x...",
    "transactions": []
  }
}
```
