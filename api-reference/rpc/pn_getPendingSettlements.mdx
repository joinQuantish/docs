## pn_getPendingSettlements

Returns all settlement transactions currently in the **pending** state — detected from the mempool but not yet confirmed in a block. This is PolyNode's core differentiator: you see settlements ~2 seconds before they appear on-chain.

### Parameters

None.

### Returns

`Array` — array of pending settlement objects.

| Field | Type | Description |
|-------|------|-------------|
| `tx_hash` | string | Transaction hash |
| `taker_token` | string | Polymarket token ID |
| `taker_side` | string | `"BUY"` or `"SELL"` |
| `taker_price` | number | Execution price (0-1) |
| `taker_size` | number | Size in USD |
| `detected_at` | number | Timestamp when detected in mempool (ms) |

### Example

```bash cURL
curl -s -X POST https://polynode.dev/rpc \
  -H "Content-Type: application/json" \
  -d '{
    "jsonrpc": "2.0",
    "method": "pn_getPendingSettlements",
    "params": [],
    "id": 1
  }'
```

### Response

```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": [
    {
      "tx_hash": "0xdef456...",
      "taker_token": "21742633...",
      "taker_side": "BUY",
      "taker_price": 0.65,
      "taker_size": 1200.00,
      "detected_at": 1709136045000
    }
  ]
}
```

<Note>
Pending settlements are retained for up to 5 minutes. Once confirmed in a block, they are removed from this set and appear in the confirmed settlements stream.
</Note>
