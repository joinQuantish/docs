## eth_call

Executes a new message call immediately without creating a transaction on the blockchain. Useful for reading contract state.

**Cache**: None (passthrough)

### Parameters

| Position | Type | Description |
|----------|------|-------------|
| 0 | `Object` | Transaction call object (`from`, `to`, `data`, etc.) |
| 1 | `QUANTITY\|TAG` | Block number (hex) or tag: `latest`, `earliest`, `pending` |

### Returns

`DATA` — return value of the executed contract call.

### Example

```bash cURL
curl -s -X POST https://api.polynode.dev/rpc \
  -H "Content-Type: application/json" \
  -d '{
    "jsonrpc": "2.0",
    "method": "eth_call",
    "params": [{
      "to": "0x2791Bca1f2de4661ED88A30C99A7a9449Aa84174",
      "data": "0x70a08231000000000000000000000000abcdef..."
    }, "latest"],
    "id": 1
  }'
```

### Response

```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": "0x0000000000000000000000000000000000000000000000000000000005f5e100"
}
```
