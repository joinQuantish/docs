---
title: "Trader Profile"
api: "GET /v2/trader/{wallet}"
description: "Full profile for any Polymarket trader."
---

Returns comprehensive stats for a trader: PnL, volume, trade count, largest win, current portfolio value, and account metadata.

<ParamField path="wallet" type="string" required>
  Ethereum wallet address (e.g., `0xc2e7800b5af46e6093872b177b7a5e7f0563be51`).
</ParamField>
