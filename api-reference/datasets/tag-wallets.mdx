---
title: "GET /tags/:slug/wallets"
description: "Paginated wallet list for a specific tag with sorting and time filtering."
---

```
GET https://datasets.polynode.dev/api/datasets/sports-esports/tags/{slug}/wallets
```

Paginated wallet list for a specific tag.

### Path Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `slug` | string | Tag slug (e.g. `nfl`, `esports`, `soccer`) |

### Query Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `after` | integer | — | Only wallets active after this unix timestamp. |
| `sort` | string | `volume` | Sort by: `volume`, `fills`, `first_seen`, `last_seen`, `markets`. |
| `limit` | integer | 50 | Results per page (max 500). |
| `offset` | integer | 0 | Pagination offset. |

### Example

```bash
curl "https://datasets.polynode.dev/api/datasets/sports-esports/tags/esports/wallets?sort=fills&limit=10"
```
