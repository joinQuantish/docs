## List Wallets

```
GET /datasets/sports-esports/wallets
```

The primary endpoint. Returns wallet addresses that traded in a specific sport category after a given date, with volume, fill count, maker/taker split, and markets traded.

### Query Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `tag` | string | — | Filter by tag slug (e.g. `sports`, `nfl`, `esports`, `soccer`). Omit for all tags. |
| `after` | integer | — | Only wallets active after this unix timestamp. |
| `before` | integer | — | Only wallets active before this unix timestamp. |
| `sort` | string | `volume` | Sort by: `volume`, `fills`, `first_seen`, `last_seen`, `markets`. |
| `limit` | integer | 50 | Results per page (max 500). |
| `offset` | integer | 0 | Pagination offset. |

### Example: All NFL wallets sorted by volume

```bash
curl "https://datasets.polynode.dev/api/datasets/sports-esports/wallets?tag=nfl&sort=volume&limit=5"
```


## Wallet Detail

```
GET /datasets/sports-esports/wallets/{address}
```

Returns a single wallet's activity broken down by every tag they traded in.

### Example

```bash
curl "https://datasets.polynode.dev/api/datasets/sports-esports/wallets/0x204f72f35326db932158cba6adff0b9a1da95e14"
```

```json
{
  "wallet": "0x204f72f35326db932158cba6adff0b9a1da95e14",
  "total_volume_usdc": 72603487.44,
  "tags_active": 1,
  "tags": [
    {
      "tag_slug": "sports",
      "fill_count": 374922,
      "total_volume_usdc": "72603487.44",
      "maker_fills": 72628,
      "taker_fills": 302294,
      "first_seen": 1761955252,
      "last_seen": 1777408494,
      "markets_traded": 52815
    }
  ]
}
```

## Whales

```
GET /datasets/sports-esports/wallets/whales
```

Wallets above a volume threshold.

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `threshold` | number | 100000 | Minimum total volume in USDC. |
| `limit` | integer | 100 | Max results. |
| `offset` | integer | 0 | Pagination offset. |

```bash
curl "https://datasets.polynode.dev/api/datasets/sports-esports/wallets/whales?threshold=1000000&limit=5"
```

## Market Makers

```
GET /datasets/sports-esports/wallets/market-makers
```

Wallets that primarily provide liquidity (high maker fill percentage).

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `min_maker_pct` | number | 80 | Minimum maker percentage (0-100). |
| `min_fills` | integer | 50 | Minimum total fills to qualify. |
| `limit` | integer | 100 | Max results. |
| `offset` | integer | 0 | Pagination offset. |

```bash
curl "https://datasets.polynode.dev/api/datasets/sports-esports/wallets/market-makers?min_maker_pct=90&min_fills=100&limit=5"
```

## New Wallets

```
GET /datasets/sports-esports/wallets/new
```

Wallets whose first-ever sports trade was within the last N days.

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `days` | integer | 7 | Look-back window in days. |
| `limit` | integer | 100 | Max results. |
| `offset` | integer | 0 | Pagination offset. |

```bash
curl "https://datasets.polynode.dev/api/datasets/sports-esports/wallets/new?days=14&limit=10"
```
