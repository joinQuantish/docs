## List Tags

```
GET /datasets/sports-esports/tags
```

Returns all available tag slugs with aggregate stats. No parameters.

```bash
curl "https://datasets.polynode.dev/api/datasets/sports-esports/tags"
```

## Tag Detail

```
GET /datasets/sports-esports/tags/{slug}
```

Returns a single tag's stats plus the top 50 wallets by volume for that tag.

## Tag Wallets (Paginated)

```
GET /datasets/sports-esports/tags/{slug}/wallets
```

Paginated wallet list for a specific tag, with sorting and time filtering.

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `after` | integer | — | Only wallets active after this unix timestamp. |
| `sort` | string | `volume` | Sort by: `volume`, `fills`, `first_seen`, `last_seen`, `markets`. |
| `limit` | integer | 50 | Results per page (max 500). |
| `offset` | integer | 0 | Pagination offset. |

```bash
curl "https://datasets.polynode.dev/api/datasets/sports-esports/tags/esports/wallets?sort=fills&limit=10"
```
