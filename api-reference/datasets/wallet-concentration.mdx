---
title: "GET /wallets/concentration"
description: "Volume concentration analysis — what share of total volume do the top N wallets hold?"
---

```
GET https://datasets.polynode.dev/api/datasets/sports-esports/wallets/concentration
```

Returns the top N wallets by volume for a tag, with each wallet's percentage of total tag volume. Reveals whether a market is whale-dominated or broadly distributed.

### Query Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `tag` | string | `sports` | Tag slug to analyze. |
| `top` | integer | 10 | Number of top wallets (max 100). |

### Example

```bash
curl "https://datasets.polynode.dev/api/datasets/sports-esports/wallets/concentration?tag=nba&top=5"
```

### Response Fields

| Field | Type | Description |
|-------|------|-------------|
| `tag_slug` | string | The tag analyzed |
| `top_n` | integer | Number of top wallets returned |
| `total_volume_usdc` | string | Total volume for this tag across all wallets |
| `top_n_volume_usdc` | string | Combined volume of the top N wallets |
| `top_n_pct` | string | Percentage of total volume held by top N wallets |
| `wallets[].wallet` | string | Wallet address |
| `wallets[].volume_usdc` | string | This wallet's volume |
| `wallets[].pct_of_total` | string | This wallet's share of total tag volume |
