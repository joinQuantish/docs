## V2 Order Structure

```
Order(
  uint256 salt,           // unique order identifier
  address maker,          // order creator wallet
  address signer,         // signing key (may differ for Safe wallets)
  uint256 tokenId,        // CT position ID (same as V1)
  uint256 makerAmount,    // amount maker provides (6 decimals)
  uint256 takerAmount,    // amount maker receives (6 decimals)
  uint8 side,             // 0=BUY, 1=SELL
  uint8 signatureType,    // 0=EOA, 1=Proxy, 2=GnosisSafe
  uint256 timestamp,      // order creation time (ms epoch)
  bytes32 metadata,       // hashed order metadata
  bytes32 builder         // builder attribution code
)
```

**Removed from V1:** `taker`, `expiration`, `nonce`, `feeRateBps`
