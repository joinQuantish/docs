---
title: "Search Markets"
openapi: "GET /v3/markets/search"
description: "Full-text search across all Polymarket markets. Returns matching markets with metadata."
---

Search market questions by keyword. Returns matching markets with metadata, volume, and resolution status.

## Request

```
GET /v3/markets/search
```

### Query parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `q` | string | required | Search query (matched against market question) |
| `limit` | integer | 100 | Max 1000 |
| `offset` | integer | 0 | Pagination offset |

## Example

```bash
curl "https://api.polynode.dev/v3/markets/search?q=bitcoin&limit=1"
```

## Response columns

| Column | Type | Description |
|--------|------|-------------|
| `condition_id` | string | Market condition ID |
| `question` | string | Market question text |
| `slug` | string | URL slug |
| `category` | string | Market category |
| `volume_all_time` | string | Total volume traded |
| `active` | boolean | Whether the market is active |
| `closed` | boolean | Whether the market is closed |
| `winning_outcome_index` | integer | Which outcome won (null if unresolved) |
| `tag_slugs` | array | Category tags |
