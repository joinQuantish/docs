---
title: "GET Market Positions (Condition)"
description: "Get all position holders for a market by condition ID. Returns wallets holding any outcome token for the condition."
---

Returns all wallets holding positions in a specific market, across all outcome tokens. Looks up token IDs from the market_tokens table using the condition ID.

## Request

```
GET /v3/markets/condition/{condition_id}/positions
```

### Query parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `sort` | string | `pnl` | Sort by: `pnl`, `amount`, `volume` |
| `limit` | integer | 100 | Max 1000 |
| `offset` | integer | 0 | Pagination offset |

## Example

```bash
curl https://api.polynode.dev/v3/markets/condition/0x00005c11.../positions?limit=10
```

### Response fields

| Field | Type | Description |
|-------|------|-------------|
| `user` | string | Wallet address |
| `token_id` | string | Outcome token ID |
| `amount` | string | Current position size (6-decimal) |
| `avg_price` | string | Weighted-average entry price |
| `realized_pnl` | string | Realized P&L |
| `total_bought` | string | Total volume traded |

Returns 404 if the condition ID doesn't match any market.
