---
title: "Builder Detail"
openapi: "GET /v3/builders/{code}"
description: "Get stats for a single builder. Fill count, maker/taker volume, fees, and unique wallet counts."
---

Returns aggregate stats for a specific builder from the `builder_stats` table. Builders are order routing services that attribute fills on Polymarket V2.

## Request

```
GET /v3/builders/{code}
```

The `code` is the builder's raw hex identifier (without `\x` prefix).

### Example

```bash
curl https://api.polynode.dev/v3/builders/4898df15ec6590495dc6c0fedf951ade3e64001d47f9caf44a64e86fc11959df
```

### Response fields

| Field | Type | Description |
|-------|------|-------------|
| `builder` | string | Builder hex code |
| `fill_count` | integer | Total attributed fills |
| `total_maker_volume` | string | Total maker-side volume (6-decimal) |
| `total_taker_volume` | string | Total taker-side volume (6-decimal) |
| `total_fees` | string | Total fees (6-decimal) |
| `unique_makers` | integer | Unique maker wallet addresses |
| `unique_takers` | integer | Unique taker wallet addresses |
| `first_fill` | string | Timestamp of earliest attributed fill |
| `last_fill` | string | Timestamp of most recent attributed fill |

Returns 404 if the builder code is not found.
