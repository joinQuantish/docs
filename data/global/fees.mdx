---
title: "Fee Events"
description: "Browse fee charge events across Polymarket trades. Filter by time range or receiver address."
---

Returns fee charge events from Polymarket trades.

## All fees

```
GET /v3/fees
```

### Query parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `after` | integer | — | Start of time range (Unix timestamp) |
| `before` | integer | — | End of time range |
| `limit` | integer | 100 | Max 1000 |
| `offset` | integer | 0 | Pagination offset |

## Fees by receiver

```
GET /v3/fees/{receiver}
```

Returns fees where a specific address received the fee payment.

### Example

```bash
curl https://api.polynode.dev/v3/fees?limit=5
```

### Response fields

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Unique fee event ID |
| `timestamp` | string | Unix timestamp |
| `receiver` | string | Address that received the fee (hex) |
| `amount` | string | Fee amount (6-decimal USDC) |
| `transaction_hash` | string | On-chain tx hash |
