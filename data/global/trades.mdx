---
title: "Global Trades"
description: "Stream through the full Polymarket trade history. Filter by time range, minimum amount, and sort direction."
---

Returns fills across all markets. Over 1.2 billion indexed. Without time filters, returns the most recent fills via the primary key index (instant).

## Request

```
GET /v3/trades
```

### Query parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `after` | integer | — | Start of time range (Unix timestamp) |
| `before` | integer | — | End of time range |
| `min_amount` | integer | — | Minimum `maker_amount_filled` (raw 6-decimal) |
| `order` | string | `desc` | `asc` or `desc` |
| `limit` | integer | 100 | Max 1000 |
| `offset` | integer | 0 | Pagination offset |

### Examples

```bash
# Latest trades
curl https://api.polynode.dev/v3/trades?limit=5

# Large trades only ($1000+ USDC)
curl https://api.polynode.dev/v3/trades?min_amount=1000000000&limit=20

# Trades in a time window
curl https://api.polynode.dev/v3/trades?after=1778000000&before=1778604000&limit=50
```

### Response fields

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Unique fill ID |
| `maker` | string | Maker wallet address |
| `taker` | string | Taker wallet address |
| `maker_asset_id` | string | Asset the maker sold (0 = USDC) |
| `taker_asset_id` | string | Asset the taker sold (0 = USDC) |
| `maker_amount_filled` | string | Maker amount (6-decimal) |
| `taker_amount_filled` | string | Taker amount (6-decimal) |
| `fee` | string | Fee (6-decimal) |
| `timestamp` | string | Unix timestamp |
| `transaction_hash` | string | On-chain tx hash |
| `builder` | string | Builder code (hex) |
| `side` | string | V2 side (0=buy, 1=sell, null for V1) |
