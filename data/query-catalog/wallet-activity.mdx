---
title: "Wallet Activity Query Catalog"
description: "Wallet activity feeds for splits, merges, neg-risk conversions, redemptions, PolyUSD flows, and fees."
---

Use these recipes when building wallet timelines, accounting views, or conditional-token activity explorers.

## Unified Non-Trade Activity

Canonical reference: [Wallet Activity](/data/wallets/activity)

```bash
curl "https://api.polynode.dev/v3/wallets/0x56687bf447db6ffa42ffe2204a05edaa20f55839/activity?limit=1" \
  -H "x-api-key: $POLYNODE_API_KEY"
```

This returns splits, merges, neg-risk conversions, and redemptions in one feed.
