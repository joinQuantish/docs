## How to Use This Catalog

Each page gives the query shape first and links back to the canonical endpoint page for full response schemas. The examples use live-tested curl shapes against `https://api.polynode.dev`.

```bash
export POLYNODE_API_KEY="..."
```

Then run any example with:

```bash
curl "https://api.polynode.dev/v3/trades?limit=1" \
  -H "x-api-key: $POLYNODE_API_KEY"
```
