---
title: "Trades Query Catalog"
description: "All major V3 trade query shapes: global, wallet, market, builder, grouped, filtered, and combo trades."
---

Use these recipes when building trade feeds, wallet activity pages, market tape views, or builder analytics.

## All Trades

Canonical reference: [Global Trades](/data/global/trades)

```bash
curl "https://api.polynode.dev/v3/trades?limit=1" \
  -H "x-api-key: $POLYNODE_API_KEY"
```

Useful filters: `after`, `before`, `token_id`, `builder`, `min_amount`, `order`, `limit`, `offset`.

## Trades by Wallet

Canonical reference: [Wallet Trades](/data/wallets/trades)

```bash
curl "https://api.polynode.dev/v3/wallets/0x56687bf447db6ffa42ffe2204a05edaa20f55839/trades?limit=1" \
  -H "x-api-key: $POLYNODE_API_KEY"
```

Useful filters: `side=maker|taker|both`, `token_id`, `condition_id`, `market_slug`, `min_amount`, `after`, `before`, `order`, `limit`, `offset`.

## Wallet Trades by Condition

Use this when a wallet dashboard needs only trades from one market condition.

```bash
curl "https://api.polynode.dev/v3/wallets/0x56687bf447db6ffa42ffe2204a05edaa20f55839/trades?condition_id=0x4f05dbc6273b89aed46bb79a961c1d8771c01925d92d439e9a81fa6241900661&limit=10" \
  -H "x-api-key: $POLYNODE_API_KEY"
```

## Wallet Trades Grouped by Order

Use this when a UI should show one visible row per order instead of every fill row.

```bash
curl "https://api.polynode.dev/v3/wallets/0x56687bf447db6ffa42ffe2204a05edaa20f55839/trades?group_by=order_hash&limit=1" \
  -H "x-api-key: $POLYNODE_API_KEY"
```

## Builder Trades

Canonical reference: [Builder Trades](/data/builders/trades)

```bash
curl "https://api.polynode.dev/v3/builders/0x4898df15ec6590495dc6c0fedf951ade3e64001d47f9caf44a64e86fc11959df/trades?limit=1" \
  -H "x-api-key: $POLYNODE_API_KEY"
```

Builder trades can also filter by `token_id`, `condition_id`, `market_slug`, `event_slug`, `tag_slug`, `category`, `side`, `min_amount`, `after`, and `before`.

## Combo Wallet Trades

Canonical reference: [Wallet Combo Trades](/data/combos/wallet-trades)

```bash
curl "https://api.polynode.dev/v3/wallets/0x63613e3b96f418332d43cd2af8dc321014d15907/combos/trades?limit=1" \
  -H "x-api-key: $POLYNODE_API_KEY"
```

To request a wallet's normal trade history plus combo trade branch:

```bash
curl "https://api.polynode.dev/v3/wallets/0x63613e3b96f418332d43cd2af8dc321014d15907/trades?include_combos=true&limit=10" \
  -H "x-api-key: $POLYNODE_API_KEY"
```
