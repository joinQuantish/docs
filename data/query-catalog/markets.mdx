---
title: "Markets Query Catalog"
description: "Search markets, resolve condition IDs, inspect token IDs, browse tags, and query market resolution data."
---

Use these recipes when building market search, market detail pages, price cards, category pages, or resolution monitors.

## Search Markets

Canonical reference: [Search Markets](/data/markets/search)

```bash
curl "https://api.polynode.dev/v3/markets/search?q=bitcoin&limit=1" \
  -H "x-api-key: $POLYNODE_API_KEY"
```
