---
title: "GET Leaderboard"
description: "Rank all 2.7 million Polymarket wallets by P&L, volume, profit, or win count. Includes realized and unrealized P&L."
---

Returns a ranked list of wallets from the `wallet_stats` materialized view. Refreshes every 30 minutes. Includes both realized and unrealized P&L.

## Request

```
GET /v3/leaderboard
```

### Query parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `sort` | string | `pnl` | Sort by: `pnl`, `volume`, `profit`, `loss`, `total`, `unrealized`, `wins`, `positions` |
| `limit` | integer | 100 | Max 1000 |
| `offset` | integer | 0 | Pagination offset |

### Sort options

| Value | What it ranks by |
|-------|-----------------|
| `pnl` | Net realized P&L (default) |
| `total` | Realized + unrealized combined |
| `unrealized` | Unrealized P&L from open positions |
| `volume` | Total trading volume |
| `profit` | Gross profit (all winning positions) |
| `loss` | Gross loss (all losing positions) |
| `wins` | Number of winning positions |
| `positions` | Total position count |

## Example

```bash
curl https://api.polynode.dev/v3/leaderboard?sort=total&limit=5
```

### Response fields (per wallet)

| Field | Type | Description |
|-------|------|-------------|
| `rank` | integer | Position on the leaderboard |
| `wallet` | string | Wallet address |
| `net_realized_pnl` | string | Net realized P&L (6-decimal) |
| `gross_profit` | string | Sum of winning positions |
| `gross_loss` | string | Sum of losing positions |
| `unrealized_pnl` | string | Paper P&L from open positions |
| `total_pnl` | string | Realized + unrealized combined |
| `wins` | integer | Winning positions |
| `losses` | integer | Losing positions |
| `position_count` | integer | Total positions |
| `total_volume` | string | Total trading volume |
| `open_positions` | integer | Currently held positions |
