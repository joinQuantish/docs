---
title: "Tag Leaderboard"
description: "P&L leaderboard filtered by tag. See the top traders in crypto, politics, sports, or any category."
---

Returns a P&L leaderboard scoped to markets under a specific tag. Aggregates realized P&L across all positions in markets matching the tag.

Works best for tags with fewer than 2,000 tokens. Very large tags (like `sports` with 500K+ markets) return an error with a suggestion to use the global leaderboard instead.

## Request

```
GET /v3/tags/{slug}/leaderboard
```

### Query parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `sort` | string | `net_pnl` | Sort by: `net_pnl`, `volume`, `profit`, `loss` |
| `limit` | integer | 100 | Max 1000 |
| `offset` | integer | 0 | Pagination offset |

### Example

```bash
curl https://api.polynode.dev/v3/tags/ufc-325/leaderboard?limit=5
```

### Response fields

| Field | Type | Description |
|-------|------|-------------|
| `wallet` | string | Wallet address |
| `net_pnl` | string | Net realized P&L in tag markets (6-decimal) |
| `gross_profit` | string | Sum of winning position P&L |
| `gross_loss` | string | Sum of losing position P&L |
| `total_volume` | string | Total volume in tag markets |
| `position_count` | integer | Number of positions in tag markets |
