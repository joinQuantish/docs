---
title: "Tag Leaderboard"
openapi: "GET /v3/tags/{slug}/leaderboard"
description: "Rank wallets by P&L within a specific market category (tag). Works for niche tags."
---

Returns a P&L leaderboard filtered to markets with a specific tag. Works for smaller/niche tags. Large tags (sports, crypto) return an error suggesting the global leaderboard instead.

## Request

```
GET /v3/tags/{slug}/leaderboard
```

### Query parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `limit` | integer | 100 | Max 1000 |
| `offset` | integer | 0 | Pagination offset |

## Example

```bash
curl https://api.polynode.dev/v3/tags/taylor-swift/leaderboard?limit=1
```

```json
{
  "columns": ["wallet", "net_pnl", "gross_profit", "gross_loss", "total_volume", "position_count"],

```

## Response fields

| Field | Type | Description |
|--------|------|-------------|
| `wallet` | string | Wallet address |
| `net_pnl` | string | Net realized P&L in this tag (6-decimal USDC) |
| `gross_profit` | string | Sum of winning positions |
| `gross_loss` | string | Sum of losing positions |
| `total_volume` | string | Total volume traded in this tag |
| `position_count` | integer | Number of positions in markets with this tag |
