---

## List all tags

```
GET /v3/tags
```

Returns all tags with the number of markets in each. Tags come from the `tag_slugs` array on market metadata, sourced from Gamma.

### Query parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `limit` | integer | 100 | Max 1000 |
| `offset` | integer | 0 | Pagination offset |

### Example

```bash
curl https://api.polynode.dev/v3/tags?limit=5
```

Response includes `tag` (slug) and `market_count` for each tag.

## Markets by tag

```
GET /v3/tags/{slug}
```

Returns all markets under a specific tag, sorted by volume.

### Query parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `sort` | string | `volume` | Sort by: `volume`, `liquidity`, `created` |
| `limit` | integer | 100 | Max 1000 |
| `offset` | integer | 0 | Pagination offset |

### Example

```bash
curl https://api.polynode.dev/v3/tags/crypto?limit=10
```

Response includes `condition_id`, `question`, `slug`, `outcomes`, `category`, `active`, `closed`, `neg_risk`, `volume_all_time`, `liquidity`, `end_date`, `event_slug`, `event_title`, `winning_outcome_index`.
