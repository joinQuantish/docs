---
title: "Wallet P&L"
openapi: "GET /v3/wallets/{address}/pnl"
description: "Get profit and loss for a wallet. Supports time-windowed P&L with ?period=1d|7d|30d|1y and tag filtering."
---

Returns P&L data for a wallet. Without a `period` parameter, returns all-time P&L. With `period`, returns realized P&L for that time window computed from individual trade events.

## Request

```
GET /v3/wallets/{address}/pnl
```

### Query parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `period` | string | -- | Time window: `1d`, `7d`, `30d`, `1y` |
| `tags` | string | -- | Comma-separated tag slugs to filter by (e.g. `politics,crypto`) |
| `market` | string | -- | Filter by condition_id |

## Examples

### All-time P&L

```bash
curl https://api.polynode.dev/v3/wallets/0xa9857c7bcb9bcfafd2c132ab053f34f678610058/pnl
```

```json
  "gross_loss": "-3936819",
  "unrealized_pnl": "91432617.7294360000000000",
  "total_pnl": "3503956",
  "position_count": 29,
  "open_positions": 29,
```

### 7-day P&L

```bash
curl https://api.polynode.dev/v3/wallets/0xa9857c7bcb9bcfafd2c132ab053f34f678610058/pnl?period=7d
```

```json

  "address": "0xa9857c7bcb9bcfafd2c132ab053f34f678610058",
```

### P&L for a specific tag category

```bash
curl https://api.polynode.dev/v3/wallets/0x.../pnl?tags=politics
```

## Response fields (all-time)

| Field | Type | Description |
|-------|------|-------------|
| `net_realized_pnl` | string | Net realized P&L (6-decimal USDC) |
| `gross_profit` | string | Sum of winning positions |
| `gross_loss` | string | Sum of losing positions (negative) |
| `unrealized_pnl` | string | Paper P&L from open positions |
| `total_pnl` | string | `net_realized_pnl + unrealized_pnl` |
| `wins` | integer | Winning position count |
| `losses` | integer | Losing position count |
| `position_count` | integer | Total positions |
| `open_positions` | integer | Currently held positions |
| `total_volume` | string | Total USDC traded |

## Response fields (with period)

| Field | Type | Description |
|-------|------|-------------|
| `realized_pnl` | string | Realized P&L in the time window (6-decimal USDC) |
| `gross_profit` | string | Sum of winning events in the window |
| `gross_loss` | string | Sum of losing events in the window (negative) |
| `wins` | integer | Winning event count |
| `losses` | integer | Losing event count |
| `events` | integer | Total P&L events in the window |
| `period` | string | The requested period |
| `after` | integer | Start timestamp (Unix) |
| `before` | integer | End timestamp (Unix) |
| `source` | string | Always `pnl_event` for time-windowed queries |
