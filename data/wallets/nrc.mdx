---
title: "Wallet NRC"
openapi: "GET /v3/wallets/{address}/nrc"
description: "Get neg-risk conversion events for a wallet. These convert between standard CTF positions and neg-risk wrapped positions in multi-outcome markets."
---

Returns neg-risk conversion events. Neg-risk markets (multi-outcome events like elections with many candidates) use a wrapper contract. This endpoint shows when a wallet converted positions through that contract.

## Request

```
GET /v3/wallets/{address}/nrc
```

### Query parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `after` | integer | — | Start of time range (Unix timestamp) |
| `before` | integer | — | End of time range (Unix timestamp) |
| `limit` | integer | 100 | Max 1000 |
| `offset` | integer | 0 | Pagination offset |

## Example

```bash
curl https://api.polynode.dev/v3/wallets/0x76a1f5f4a6ac99f5de9fba514d609072fc7c4652/nrc?limit=2
```

{

### Response fields

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Unique conversion event ID |
| `timestamp` | string | Unix timestamp |
| `neg_risk_market_id` | string | The neg-risk parent market ID |
| `amount` | string | Amount converted (6-decimal USDC). `5000000` = $5.00 |
| `index_set` | string | Which outcome index was converted |
| `question_count` | integer | Number of outcomes in the neg-risk market |
