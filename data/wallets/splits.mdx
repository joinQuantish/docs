---
title: "Wallet Splits"
description: "Get all CTF token split events for a wallet. Splits convert USDC collateral into a complete set of outcome tokens."
---

Returns split events where the wallet deposited USDC collateral and received a full set of outcome tokens for a condition.

## Request

```
GET /v3/wallets/{address}/splits
```

### Query parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `after` | integer | — | Start of time range (Unix timestamp) |
| `before` | integer | — | End of time range (Unix timestamp) |
| `limit` | integer | 100 | Max 1000 |
| `offset` | integer | 0 | Pagination offset |

## Examples

### Recent splits

```bash
curl https://api.polynode.dev/v3/wallets/0x56687bf4.../splits?limit=5
```

### Splits in a time window

```bash
curl .../v3/wallets/0x.../splits?after=1778000000&before=1778604000
```

## Response fields

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Unique split event ID |
| `stakeholder` | string | Wallet address |
| `condition_id` | string | Condition split into |
| `amount` | string | USDC collateral deposited (6-decimal) |
| `timestamp` | string | Unix timestamp |
| `transaction_hash` | string | On-chain transaction hash |
