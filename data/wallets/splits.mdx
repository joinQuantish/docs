---
title: "Wallet Splits"
description: "Get position split events for a wallet. Splits deposit USDC collateral and receive a complete set of outcome tokens for a market."
---

Returns split events where the wallet deposited USDC and received outcome tokens. Each split references a condition (market) and the collateral amount.

## Request

```
GET /v3/wallets/{address}/splits
```

### Query parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `after` | integer | — | Start of time range (Unix timestamp) |
| `before` | integer | — | End of time range (Unix timestamp) |
| `limit` | integer | 100 | Max 1000 |
| `offset` | integer | 0 | Pagination offset |

## Example

```bash
curl https://api.polynode.dev/v3/wallets/0xada200001000ef00d07553cee7006808f895c6f1/splits?limit=2
```

```json
{
  "rows": [
    [
      "0x48c69f12...1b77b5_0x4bc",
      "1778492423",
      "0xd0dbdc94...18d5e7",
      "2830000"
    ]
  ],
  "columns": [
    {"name": "id", "type": "text"},
    {"name": "timestamp", "type": "text"},
    {"name": "condition", "type": "text"},
    {"name": "amount", "type": "text"}
  ],
  "rows_returned": 2,
  "has_more": true,
  "elapsed_ms": 4
}
```

### Response fields

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Unique split event ID |
| `timestamp` | string | Unix timestamp |
| `condition` | string | Condition ID of the market |
| `amount` | string | USDC collateral deposited (6-decimal). `2830000` = $2.83 |
