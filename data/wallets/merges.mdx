---
title: "Wallet Merges"
description: "Get position merge events for a wallet. Merges return a complete set of outcome tokens and receive USDC collateral back."
---

Returns merge events where the wallet returned outcome tokens and received USDC collateral. The inverse of a split.

## Request

```
GET /v3/wallets/{address}/merges
```

### Query parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `after` | integer | — | Start of time range (Unix timestamp) |
| `before` | integer | — | End of time range (Unix timestamp) |
| `limit` | integer | 100 | Max 1000 |
| `offset` | integer | 0 | Pagination offset |

## Example

```bash
curl https://api.polynode.dev/v3/wallets/0xada100874d00e3331d00f2007a9c336a65009718/merges?limit=2
```

```json
{
  "rows": [
    [
      "0x56e84b69...e4a1f4_0x612",
      "1778482999",
      "0x649340b2...375e34",
      "20000000"
    ]
  ],
  "columns": [
    {"name": "id", "type": "text"},
    {"name": "timestamp", "type": "text"},
    {"name": "condition", "type": "text"},
    {"name": "amount", "type": "text"}
  ],
  "rows_returned": 2,
  "has_more": true,
  "elapsed_ms": 7
}
```

### Response fields

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Unique merge event ID |
| `timestamp` | string | Unix timestamp |
| `condition` | string | Condition ID of the market |
| `amount` | string | USDC collateral received (6-decimal). `20000000` = $20.00 |
