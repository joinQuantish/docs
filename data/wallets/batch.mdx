---
title: "Batch Wallet P&L"
description: "Query P&L for up to 100 wallets in a single request. Returns realized P&L, unrealized P&L, win/loss record, and volume for each wallet."
---

Query multiple wallets at once. Returns the same summary data as the single wallet endpoint, but for up to 100 wallets in one call. Responses typically return in under 5ms regardless of how many wallets you query.

## Request

```
POST /v3/wallets/batch
```

### Request body

```json
{
  "wallets": [
    "0x56687bf447db6ffa42ffe2204a05edaa20f55839",
    "0x204f72f35326db932158cba6adff0b9a1da95e14",
    "0xdb27bf2ac5d428a9c63dbc914611036855a6c56e"
  ]
}
```

| Field | Type | Description |
|-------|------|-------------|
| `wallets` | array | Up to 100 wallet addresses |

## Example

```bash
curl -X POST https://api.polynode.dev/v3/wallets/batch \
  -H 'Content-Type: application/json' \
  -d '{"wallets":["0x56687bf447db6ffa42ffe2204a05edaa20f55839","0x204f72f35326db932158cba6adff0b9a1da95e14"]}'
```

## Use cases

- **Leaderboard building** — pull P&L for a custom watchlist of wallets
- **Copy trading dashboards** — monitor multiple traders simultaneously
- **Whale tracking** — check 50+ wallets every few seconds for position changes
- **Portfolio aggregation** — sum P&L across multiple wallets belonging to the same entity
