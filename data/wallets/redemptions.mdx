---
title: "GET Wallet Redemptions"
description: "Get redemption events for a wallet. Redemptions pay out USDC when a resolved market's winning tokens are claimed."
---

Returns redemption events where the wallet claimed USDC from resolved markets. Each redemption shows the condition, payout amount, and which outcome index sets were redeemed.

## Request

```
GET /v3/wallets/{address}/redemptions
```

### Query parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `after` | integer | — | Start of time range (Unix timestamp) |
| `before` | integer | — | End of time range (Unix timestamp) |
| `limit` | integer | 100 | Max 1000 |
| `offset` | integer | 0 | Pagination offset |

## Example

```bash
curl https://api.polynode.dev/v3/wallets/0x56687bf447db6ffa42ffe2204a05edaa20f55839/redemptions?limit=2
```


### Response fields

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Unique redemption event ID (tx_hash + log index) |
| `timestamp` | string | Unix timestamp of the redemption |
| `condition` | string | Condition ID of the resolved market |
| `payout` | string | USDC payout received (6-decimal). Divide by 1,000,000 for USD. |
| `index_sets` | array | Outcome index sets that were redeemed |

The payout of `120469388830` in the example equals **$120,469 USD**.
